## Installation

1. Open **Extensions** sidebar panel in VS Code. `View â†’ Extensions`
2. Search for `Bits Theme`
3. Click **Install**
4. Click **Reload** to reload VS Code
5. Go to `File > Preferences > Theme > Color Theme` and select **Bits Theme**
